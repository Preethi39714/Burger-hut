# 🍔 Burger Hut

A simple food ordering web application built using Python and Flask.

## Features

* View menu items
* Place orders
* Basic user interface

## Tech Stack

Python, Flask, HTML, CSS

## Run Locally

```bash
git clone https://github.com/Preethi39714/Burger-hut.git
cd Burger-hut
pip install -r requirements.txt
python app.py
```
